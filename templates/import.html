📥 导入 CSV

使用说明

  • CSV 第一行为表头，支持中文字段名（如 用户、部门、CC邮箱、邮箱密码、VPS的IP 等），系统会自动识别
  • 编码支持 UTF-8 (含/不含 BOM) 和 GBK（Excel "另存为 CSV" 默认就是 GBK）
  • 追加模式：在现有数据后增加；替换模式：先清空所有账号和封号历史，再导入（危险，慎用）
  • 必填：用户姓名（或 用户、姓名）。其他字段缺省时取默认值
  • 日期格式支持：2025-05-31 / 2025/5/31 / 5月31日
  • 布尔字段识别为"真"的值：是 / yes / true / 1 / 已交付 / 已安装 / ✓
  • 导入后密码会自动加密入库
